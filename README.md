### 使用插件

- | 插件名称          | 作用                | tip                                                    |
  | ----------------- | ------------------- | ------------------------------------------------------ |
  | @inquirer/prompts | 命令行交互工具      |                                                        |
  | commander         | 自定义命令行指令    |                                                        |
  | chalk             | 命令行美化工具      | 例如对重点信息添加颜色                                 |
  | ora               | 命令行 loading 动效 |                                                        |
  | cross-spawn       | 跨平台 shell 工具   | 在脚手架里面，可以用来自动执行 shell 命令 （安装依赖） |
